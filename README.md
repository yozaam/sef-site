## Run Locally

- Clone your forked repository
    ```
    git clone https://github.com/USERNAME/sef-site
    cd sef-site
    ```
- Install all the dependecies
    ```
    npm install
    ```
- Start the server with 
    ```
    npm start
    ```

- Visit your app at http://localhost:9000
